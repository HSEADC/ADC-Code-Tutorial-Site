Как сделать слайдер

Слайдер — это интерактивный элемент, в котором большая часть контента скрыта, а показывается только один элемент за одно время. Для его создания нужно уметь скрывать элементы и управлять положением элементов на экране.легко исправить, если спрятать скролл.

Как сделать бургер-меню

На многих сайтах при нажатии на иконку бургера выпадает меню в сайдбаре, а бургер переходит в крестик, при нажатии на который, меню закрывается. Достаточно простая вещь, но есть много нюансов.

Как работать с селектом

Селекты постоянно встречаются в пользовательских интерфейсах, обычно в открытых билиотеках они или недостаточно функциональны или не подходят по дизайну. Расскажем о видах селектов и их особенностях.

Superorganism

Это самые большие структуры, в которых организовываются все остальные. Этот раздел содержит несколько примеров комбинации и взаимодействий разных по строению компонентов.

Sing in

Стандартная форма входа в какое-либо приложение или сервис.

Modal Alert

Всплывающее модальное интерактивное окно различного назначения.

Organism

Это компоненты, которые состоят из групп молекул, объединенных вместе, образуют относительно сложный, отдельный участок интерфейса.

Slider

Распространенный вариант хранения фото на сайте или в приложении.

Select

Интерактивный компонент, который предназначен для управления данными.

Multiselect Input

Расширенный функционал Select, который продполагаем множественный выбор.

Menu

Стандрартный компонент навигации.

Select Input

Стандартный селект едничного выбора элемента списска.

Molecule

Молекулы — это группы атомов, связанных друг с другом, и являются наименьшими функциональными единицами интерфейса. Как правило, молекулы берут на себя индивидуальние свойства и служат фундаментом систем проектирования.

Slider Content

Функциональный элемент слайдера.

Atom

Атомы являются основными строительными блоками интерфейсов. Атомы — это наши HTML-теги, такие как метка формы, вход или кнопка. Они также могут включать в себя более абстрактные элементы, такие как цветовые палитры и шрифты.

Button

Стандартная кнопка действия.

Chip

Интерактивный элемент множественного выбора.

Option Item

Функциональный элемент выпадающего списка.

Menu Item

Интерактивный элемент коллекции меню.

Input

Функциональный элемент взяимодействия с интерфейсом.

Icon

Стандартные иконки.

О Нас

ADC Code Tutorial — это веб-учебник с туториалами для изучения продвинутой веб–вёрстки, разработанный в рамках зонтичного бренда кафедры «Дизайн и программирование» в Школе Дизайна НИУ ВШЭ.

Этот веб-учебник появился как дипломный проект студентки профиля Филачевой Полины. Куратором работы является руководитель направления Захар День.

Нашей задачей является собрать личный опыт из принципов и фишек по веб-версте в одном месте, чтобы упростить процесс перехода от простых статичных сайтов к интерактивным интерфейсам.
