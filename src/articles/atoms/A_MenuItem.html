Chip

Chip помогает людям выводить нужную информацию,фильтровать содержимое или инициировать действия. Chip могжет отображать разные типы интерактивных элементов в одной области, таких как список выбираемого времени фильма или серия контактов электронной почты.

Структура компонента

Chip состоит из контейнера, текстового lable и ,иногда, из одной или нескольких иконок. Так как выбираемая строка из списка может быть длинной, мы можем задать максимальный размер контейнера chip и срезать выбранный текст. Делаем мы это для того, чтобы chip в input вставали корректно.

Состояние компонента

Управлять выбранными чипсами можно из самого input , если chip имеет собственную кнопку удаления, или отменять выбор через выпадающий option list . Состояния chip при различных условиях покаказаны ниже.

Реализация

Для начала в папке с компонентами создаем новую папку A_Chip, в неё кладем A_Chip.jsx и A_Chip.scss. Затем описываем функционал chip. Получаем следующий код, который описывает все состояния disabled, initial, hover.

Chip содержит icon Q_Icon cross, которая лежит в отдельной папке image в файле Q_Icon.scss. В нем мы храним все иконки, которые нужны для различных компоннтов, а их фон и размер задаем в файле со стилем всего chip. Это мы покажем в структуре SCSS. 

import classnames from 'classnames'
import React, { PureComponent } from 'react'
import './A_Chip.scss'

export default class A_Chip extends PureComponent {
  constructor(props) {
    super(props)
  }

  handleClick = () => {
    const { disabled, handleClick } = this.props

    if (!disabled) {
      handleClick()
    }
  }

  render() {
    const { text, disabled } = this.props

    const classes = classnames({
      A_Chip: true,
      disabled: disabled
    })

    return (
    <div className={classes}>
        {text}
        <div className="Q_Icon cross" onClick={this.handleClick} />
      </div>
    )
  }
}

Выводим получившуюся структуру в файле с примером: 

import React from 'react'
import ReactDOM from 'react-dom'
import A_Chip from '../../components/atoms/A_Chip/A_Chip.jsx'
import './example_4.scss'

function handleClick() {
  console.log('click')
}

document.addEventListener('DOMContentLoaded', () => {
  ReactDOM.render(
    <>
      <A_Chip text="Text" handleClick={handleClick} />
      <A_Chip text="Text" disabled={true} handleClick={handleClick} />
    </>,
    document.body.appendChild(document.createElement('div'))
  )
})

Затем прописываем стили для A_Chip и Q_Icon cross. Не забываем описать для chip состоянии disabled и hover.

  padding: 6px 38px 6px 10px;
  border-radius: 10px;
  font-size: 16px;
  font-weight: 300;
  color: #426060;
  background-color: #b7fcfc;
  cursor: default;
  user-select: none;

  .Q_Icon.cross {
    position: absolute;
    top: 6px;
    right: 10px;
    width: 20px;
    height: 20px;
    margin-left: 8px;
    border-radius: 20px;
    background-color: #d7ffff;
    cursor: pointer;

    &:hover {
      background-color: #5be7e7;
    }
  }

  &.disabled {
    color: #abd9d9;
    background-color: #f2ffff;
    cursor: default;

    .Q_Icon.cross {
      &:hover {
        background-color: #d7ffff;
        cursor: default;
      }
    }
  }
}
