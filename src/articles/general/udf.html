Проектирование компонентов в изоляции

Это подход основываетя на применении компонентно ориентированого программирования. Попросту говоря, разработка, основанная на компонентах, представляет собой проектирование приложений посредством создания слабо связанных друг с другом независимых блоков кода. У каждого из них есть интерфейс, предназначенный для организации взаимодействия с другими частями системы. Множество компонентов, объединённых друг с другом посредством композиции, формируют модульное приложение.

В данном примере мы рассмотриваем особый принцип верстки простого компонента в изоляции.

Пример

Для начала создаем кнопку. 

<div class=”button”></div>

CSS код у этой кнопки выглядит так. 

.button {
width: 20px;
height: 20px;
border-radius: 10px;
background-color: #d7ffff;
background-image: url(“cross.svg”);
background-size: 8px 8px;
background-position: center;
}

Мы имеем div с классом button и определенными параметрами, выглядит он так.

Затем, эту кнопку мы хотим вывести в другом компоненте, например в chip. HTML код чипса выглядит так. Про chip можно изучить в этом туториале. 

<div class=”chip”>
  Something
  <div class=”button”></div>
  </div>
}

Дилема заключается в том, что мы можем CSS код положить и в тот компонет и в другой. Следуя принципу проектирования компонентов в изоляции, малый компонет не должен ничего знать о среде, которая на него влияет. Код о приобретённых свойствах малого компонента помещается в больший компонент. 

.chip {
position: relative;
height: 32px;
padding: 0 32px 0 12px;
border-radius: 16px;
font-family: Helvetica, Arial, sans-serif;
font-size: 14px;
line-height: 32px;
background: #D3FFFF;
}

.chip .button {
position: absolute;
top: 6px;
right: 6px;
}

Теперь мы имеем div с классом chip, в котором есть текст и div с нашей кнопкой.

Следуя этому принципу и далее, мы создадим список из этих элементов. 

<div class=”chip”>
  Something
  <div class=”button”<</div>
</div>

<div class=”chips”>
 <div class=”chip”>
  Something
  <div class=”button”></div>
</div>

<div class=”chip”>
  Something
  <div class=”button”></div>
</div>

<div class=”chip”>
  Something
  <div class=”button”></div>
</div>
</div>

CSS код будет выглядеть так. 

.chips {
display: flex;
padding: 16px;
background: #FFFFFF;
box-shadow: 0px 0px 35px rgba(0, 0, 0, 0.11);
border-radius: 10px;
}

.chips .chip {
margin: 0 8px 8px 0;
}
